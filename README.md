# 🚀 PortSyncro - Smart Portfolio Synchronization

> **The Portfolio Synchronizer That Solves Real Investment Problems**

Ever found yourself struggling with the same stock (like BBRI) across 3 different brokerages, or Bitcoin spread across 2 different exchanges? Constantly calculating your true average price and overall portfolio performance?

**PortSyncro** is the solution to portfolio fragmentation - a modern, secure platform that unifies your investments across multiple platforms into one intelligent dashboard.

## 🎯 The Problem We Solve

### The Challenge
- **Same assets across multiple brokerages/exchanges** - No unified view
- **Manual average price calculations** - Prone to errors and time-consuming
- **Fragmented portfolio tracking** - Difficulty monitoring overall performance
- **Platform downtime risks** - R*B Sekuritas, In*o*ax incidents highlight the need for diversification
- **No real-time synchronization** - Outdated information leads to poor decisions

### The Solution
PortSyncro bridges the gap between smart diversification and manageable portfolio tracking.

## ✨ Key Features

### 🎯 Core Functionality
- **🔄 Portfolio Synchronization** - Unify assets across multiple platforms
- **💰 Smart Average Price Calculator** - Accurate calculations for multiple purchases
- **📊 Real-time Performance Tracking** - Live updates from Yahoo Finance & crypto APIs
- **💱 Dual Currency Support** - Gain/loss calculations in both IDR and USD
- **📈 Comprehensive Analytics** - Detailed performance insights and reporting

### 🛡️ Security & Reliability
- **Firebase Authentication** with secure session management
- **Protected routes** with automatic redirects
- **Input validation & sanitization** for XSS protection
- **Rate limiting** (30 requests/minute per user/IP)
- **AES-256-CBC encryption** for sensitive data
- **Security headers** (CSP, HSTS, X-Frame-Options)
- **Real-time threat detection** and monitoring

### 🌍 Market Coverage
- **📈 Indonesian Stocks (IDX)** - Yahoo Finance API integration
- **🪙 Global Cryptocurrencies** - CryptoCompare API support
- **💱 Exchange Rates** - Real-time USD/IDR conversion
- **🌐 Multi-language** - English and Indonesian interface
- **📱 Responsive Design** - Works seamlessly on all devices

## 🛠️ Tech Stack

- **Frontend**: Next.js 15, React 18, Tailwind CSS
- **Backend**: Firebase (Authentication, Database)
- **APIs**: Yahoo Finance, CryptoCompare, Exchange Rate API
- **Security**: AES-256 encryption, rate limiting, input validation

## 📊 Data Sources

- **Stocks**: Yahoo Finance API (Indonesia Stock Exchange)
- **Cryptocurrencies**: CryptoCompare API (Global crypto data)
- **Exchange Rates**: Exchange Rate API (USD/IDR conversion)

## 🚀 Why This Matters

In today's volatile market, having your investments spread across multiple platforms is smart risk management. But without proper tools, it becomes a nightmare to manage. PortSyncro gives you the security of diversification without the complexity of manual tracking.

This project represents months of development and real-world problem-solving. It's not just another portfolio tracker—it's a solution born from actual investment challenges that many of us face.
